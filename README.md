# Ping-Pong Game
A simple browser-based Ping-Pong game developed using HTML, CSS, and JavaScript. Features both desktop and touch controls.

## Features

- **Responsive Canvas:** Adapts to any screen size.
- **Player vs Bot:** Control the paddle at the bottom while the bot controls the top paddle.
- **Desktop & Mobile Controls:** Arrow keys or `A/D` for desktop; touch zones for mobile devices.
- **Dynamic Ball Speed:** The ball gradually increases in speed over time.
- **Score Tracking:** Keeps track of player and bot scores.
- **Simple UI:** Play button and game info displayed before starting.

## How to Play

1. Open `index.html` in a browser.
2. Click the **PLAY** button to start the game.
3. Move the paddle left/right using:
   - **Desktop:** Left and Right Arrow keys or `A/D`
   - **Mobile:** Tap the left or right side of the screen
4. Try to hit the ball and score points while preventing the bot from scoring.
